1. Что такое и зачем нужен HTML?

ХТМЛ это язык написания сайтов в наших этих интернетах (язык гипертекстовой разметки). Нужен для написания страничек в интернете.

2.Что такое и зачем нужен Git?

Гит - как говорится в интернете система котнтроля версий. В общем через нее я отправляю свои работки в облако, где так же можно проследить историю изменений

3.Что такое SEO?

комплекс мер по оптимизации сайта для поднятия сайта в поиске

4.В чем отличие Git от GitHub?

Гит - система управления версиями, гитхаб - веб-ресурс, на котором эти версии хранятся

5.Что такое тег? Что такое атрибут?

Тег - это наши строчки, при помощи которых мы создаем сайты, основа языка ХТМЛ. Атрибут - настройка этого тега, цвет, размер, запах, вкус

6.Чем отличается язык разметки от языка програмирования? HTML можно назвать языком программирования? Аргументируйте свою позицию.

Нет, т.к. не является полным по Тьюрингу - при помощи него нельзя записать любую вычисляемую функцию

7.Зачем нужен атрибут alt у изображений?

Атрибут алт у изображений нужен для того, чтобы в случае, если наш сайт просматривает человек с ограниченными возможностями он понимал что там нарисовано.

1.Зачем нужен тег < label>? Как объединить < label> и < input>?

Для названия и для связи элемента формы (input) и элемента label, связать их можно при помощи id и for, или вложить input в lable

2. Как объединить радиокнопки в группу?

присвоить им одинаковый атрибут name.

3.Какие условия нужно соблюсти, чтобы отправить файл на сервер?

нужно указывать акцепт, нужно указывать enctype, нужно указывать тип файла

4.Что такое HTTP? Какие существуют HTTP методы?

Протокол для передачи гипертекстовых документов. Методы ХТТП - POST - создать, DEL - удалить, PUT обновить, GET прочитать

5.Зачем используется атрибут name?

задает имя для сервера или якорей

1.Что такое семантика? Зачем нужна?

Семантика - это набор тегов для улучшения структуры веб-страницы, которые добавляют смысловое значение тексту в тегах

2.В каких случаях используется тег < article>?

Когда надо добавить каку-либо самостоятельную статью в блок

3.Зачем используются ARIA-роли и ARIA-атрибуты?

ARIA - роли и атрибуты, которые делают сайт более доступным для людей с ограниченными возможностями. Роли - указывают назначение элемента, его роль, ариа-атрибуты делятся на 2 группы: атрибуты состояния (меняются при взаимодействии с пользователем) и атрибуты свойства (не изменяются)

4.Зачем используются теги < iframe>, < audio>, < video>?

Тег < iframe> создает плавающий фрейм, который находится внутри обычного документа, он позволяет загружать в область заданных размеров любые другие независимые документы. Тег audio и video добавляет, воспроизводит и управляет настройками аудиозаписи/видеозаписи на веб-странице.

За что отвечает свойство display?

Чем отличается width: 100% от width: auto?

если задать элементу ширину 100% то он займет всю доступную ширину в родительском блоке не учитывая отступы (отступы выпадут из блока), а если auto то ширина дочернего блока автоматически уменьшится за счет отступов

Зачем нужно свойство box-sizing?

Чем отличаются свойства justify-content и align-items?

Что такое Flexbox и CSS Grid?

набор свойств в CSS для построения сеток

Как строить сетки на флоатах (float)? Для чего должно использоваться свойство float?

чтобы построить внятную сетку пригодятся: свойство clear:left следующему блоку или overflow:auto родителю, невидимый div со свойством clear:both или присвоить родителю флоат. Используется свойство флоат если у вас задача сделать сетку для ИЕ 6 или для красивого обтекания текстом

Чем отличается Flexbox свойство justify-content от align-items?

justify-content выравнивает элементы по главной оси, а align-items по поперечной

Зачем используется Flexbox свойство flex-basis?

Чем отличается flex-grow от flex-shrink?

Чем отличаются CSS Grid свойства justify-items, justify-content и justify-self?

justify-content выравнивает сетку, justify-items выравнивает содержимое, justify-self – отдельный элемент
